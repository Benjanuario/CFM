1. Informações que Coletamos

A CFM está comprometida com a transparência na coleta de dados. Coletamos apenas informações necessárias para o funcionamento do serviço:

Dados Não-Pessoais

  • Nome de utilizador gerado automaticamente
  • Região de acesso (cidade/província)
  • Tipo de dispositivo e navegador
  • Horário de acesso
  • Preferências de idioma

Conteúdo Gerado

  • Mensagens enviadas no chat
  • Interações com atualizações
  • Preferências de notificação
  • Configurações salvas localmente

Fluxo de Dados

1
Geração automática de identificador anônimo
2
Armazenamento local no seu dispositivo
3
Transmissão encriptada para servidores seguros (Firebase / Google App Script
4
Processamento anonimizado para estatísticas
Importante: Não coletamos dados pessoais sensíveis como BI, NUIT, NIB, endereços ou informações financeiras. Seus dados de navegação são anonimizados.

2. Finalidade do Tratamento

Utilizamos os dados coletados exclusivamente para:

2.1. Funcionalidade do Serviço

  • Fornecer actualizações em tempo real sobre horários
  • Facilitar comunicação entre passageiros/usuários
  • Personalizar a experiência de uso
  • Manter histórico de mensagens no chat

2.2. Melhoria do Serviço

  • Análise estatística anônima de uso
  • Identificação de problemas técnicos
  • Otimização de performance
  • Desenvolvimento de novas funcionalidades

2.3. Segurança

  • Prevenção de abuso e fraudes
  • Detecção de actividades suspeitas
  • Proteção da integridade do sistema
  • Cumprimento de obrigações legais

3. Compartilhamento de Dados

5. Medidas de Segurança

Implementamos medidas técnicas e organizacionais robustas para proteger seus dados:

Encriptação SSL/TLS: Todas as comunicações são encriptadas em trânsito
Armazenamento Seguro: Dados armazenados em servidores com certificação do Google /Firebase Hosting
Acesso Restrito: Apenas pessoal autorizado tem acesso aos dados
Backups Regulares: Cópias de segurança diárias com retenção controlada
Monitoramento Contínuo: Os dados são monitorados para evitar a perda sem BackUp
Pseudonimização: Dados são anonimizados sempre que possível

8. Retenção de Dados

Mantemos seus dados apenas pelo tempo necessário:

8.1. Períodos de Retenção

  • Dados de actividade: 30 dias
  • Logs de acesso: 30 dias
  • Mensagens de chat: 94 horas
  • Dados anonimizados: Indefinido (para estatísticas)

8.2. Critérios de Eliminação

  • Cumprimento do prazo de retenção
  • Solicitação do titular
  • Encerramento da conta
  • Finalidade alcançada
